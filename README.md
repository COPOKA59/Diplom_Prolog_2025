# Web-приложение для писателей «Пролог»
Приложение для написания текстовых произведений. Позволяет создавать, редактировать и удалять свои литературные работы, а также читать опубликованные тексты других зарегистрированных авторов.

Включает в себя возможность добавления произведений в сборники пользователя для организации работ и упрощённого доступа к ним.

## Используемые технологии
### Backend
* [Django](https://www.djangoproject.com/) - фреймворк для серверной части.
* [Django REST framework](https://www.django-rest-framework.org/) - фреймворк для реализации REST API.
* [PostgreSQL](https://www.postgresql.org/) - база данных.

### Frontend
* [Vite](https://vite.dev/) - система сборки.
* [Vue.js](https://vuejs.org/) - фреймворк для клиентской части.
* [PrimeVue](https://primevue.org/) - библиотека UI-компонентов.
* [Vuetify](https://vuetifyjs.com) - библиотека для вёрстки и UI-компонентов.
* [Axios](https://axios-http.com/) - библиотека для работы с HTTP-запросами.
* [Pinia](https://pinia.vuejs.org/) - библиотека для управления состоянием.
